# Event Planner

### 1. Описание инструмента
`itertools` — это стандартная библиотека Python, содержащая функции для создания эффективных итераторов. Они позволяют обрабатывать данные "лениво", экономя оперативную память.

### 2. Цель изучения
Освоить комбинаторные функции (`product`, `combinations`), бесконечные итераторы (`cycle`) и продвинутую итерацию данных (`zip_longest`, `islice`).

### 3. Описание реализованной задачи
Программа автоматизирует планирование банкета:
- Генерация полного меню (декартово произведение).
- Формирование уникальных пар гостей.
- Создание циклического графика дежурств.
- Распределение гостей по столам с учетом нехватки мест.
- Поиск комбинаций грузов заданной массы.

### 4. Инструкция по запуску
1. Убедитесь, что установлен Python 3.x.
2. Скачайте файл `event_planner.py`.
3. Запустите команду: `python event_planner.py`.

### 5. Общие выводы
Использование `itertools` сокращает количество вложенных циклов `for` и делает код более читаемым и производительным при работе с большими объемами данных.
